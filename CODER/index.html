Quick Reference

Commands

Coder create task <desc>Create story in BACKLOG
Coder planPlan stories in PLAN column
Coder executeImplement stories in EXECUTION
Coder move S001 to PLANMove story between columns
Coder add to S001 <text>Append to USER PROMPT
Coder for S001 <instr>Add instruction directly
Coder statusBoard summary
Coder update memoryRe-index knowledge base
Coder <bug desc>Smart bug detection

Workflow

BACKLOG → PLAN → REVIEW → EXECUTION → TESTING → DONE
Human moves stories forward. Coder moves to REVIEW (after plan) and TESTING (after execute). Only Human can move to DONE.
